\# 🚀 AI 媒体处理工具箱



一体化解决动漫视频处理、AI打标、标签筛选、OP/ED识别、磁力下载等需求。



\## ✨ 功能亮点



\- 🎬 视频帧提取（支持批量/单视频）

\- 🏷️ AI 自动打标（WD-ViT-TAGGER-v3）

\- 📄 智能标签筛选（支持模糊/精确/AND/OR 匹配）

\- 🖼️ 图像标签编辑器（支持 undo/redo/批量操作）

\- 📺 米咔动漫更新追踪 + 磁力链接下载（支持 aria2 / qBittorrent）

\- 🎞️ 下载后直接播放（MKV 自动转 MP4 兼容播放）
